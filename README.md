# 客栈管理系统 java1203

## 项目概述

客栈管理系统是一个基于前后端分离架构的开源项目，使用Vue3、TypeScript和Express等技术栈，实现了对客栈日常业务的高效管理。

## 技术栈

- 前端框架：Vue3
- 前端路由管理：vue-router
- 后端框架：Express
- 数据库：MySQL

## 功能模块

### 系统角色

- 普通管理员
- 超级管理员

### 界面概览

- 管理员登录界面
- 管理员界面
  - 房型管理界面
  - 客房管理界面
  - 住户管理界面
  - 账单界面

## 系统运行环境

- 前端：现代浏览器环境
- 后端：Node.js 运行环境
- 数据库：MySQL 数据库环境

## 部署步骤

1. 搭建数据库环境，导入数据模型与初始数据。
2. 部署后端服务，配置好环境变量，并启动服务。
3. 部署前端项目，配置好接口地址，并构建静态资源。

## 目录结构

```
.
├── client        // 前端项目目录
│   ├── public
│   └── src
└── server        // 后端项目目录
    ├── routes
    ├── models
    ├── controllers
    └── middleware
```

## 核心亮点

- 使用Vue3和TypeScript提升项目质量和开发效率。
- 基于Express的后端服务易于维护和扩展。
- 提供丰富的管理界面，满足客栈日常业务需求。
- 开源性质，允许用户根据自身需求进行定制化修改。
